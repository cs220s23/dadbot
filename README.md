## Installation

### Local

1. Clone this repo and enter the directory

```bash
git clone REPONAME
cd dadbot
```

2. Create a virtual environment and install requirements

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
